# UE Data Simulation Replay

## Description
Offline replay of simulated trajectory data from CSV file using Unreal Engine.

## Features
- CSV parsing (C++)
- Time-based trajectory playback
- Trajectory visualization
- Play / Pause / Scrub UI
- Simulation replay use case

## Use cases
- Training replay
- Simulation analysis
- Data visualization

## Tech
- Unreal Engine (C++)
- UMG
- Git versioning
